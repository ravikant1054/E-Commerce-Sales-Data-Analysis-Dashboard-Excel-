----------------------------------------------------
Project Overview
--------------
Yeh project Max Store ke 2022 ke sales data ka ek detail analysis hai. Isme Excel ka use karke data cleaning, processing aur visualization ki gayi hai taaki business performance ko samjha ja sake aur key insights nikale ja sakein.

Key Insights (Main Highlights)
---------------------------------

Top State: Sabse zyada sales Maharashtra (0.07M) se aayi hain.

Gender Factor: Lagbhag 99% customers Mahilaon (Women) hain, jo dikhata hai ki store women-centric hai.

Target Age Group: Sales mein sabse bada hissa Adults (80.1%) ka hai.

Best Channels: Sabse zyada orders Amazon (34.5%), Myntra, aur Flipkart se mil rahe hain.

Delivery Success: 93% orders successfully deliver ho chuke hain.

Tools Used
--------------
Microsoft Excel: Data cleaning aur Pivot Tables ke liye.

Data Visualization: Bar charts, Pie charts, aur Slicers ka use karke interactive dashboard banaya gaya.

 Dashboard Components
 -------------------------
Maine is project mein niche diye gaye parameters ko analyze kiya hai:

Sales & Order by Month: Mahine ke hisaab se revenue aur orders ka trend.

Gender-wise Contribution: Men vs Women sales distribution.

Order Status: Deliver, Cancelled, Returned, aur Refunded orders ka percentage.

Top 5 States: Woh states jo sabse zyada revenue generate kar rahi hain.

Age Group Analysis: Teenagers, Adults, aur Seniors ka sales contribution.

Channel Analysis: Amazon, Myntra, Flipkart jaise platforms ki performance.

How to use
-------------
Repository ko clone ya download karein.

Excel file ko open karein.

Slicers ka use karke data ko filter karein (Jaise Category, Channel, ya Month) aur live updates dekhein.
